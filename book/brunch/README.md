# MHMbook 브런치 연재 계획

기준일: 2026-07-05

## 목표

`나는 딴 돈의 절반만 가져가`의 장별 원고를 책 순서대로 브런치 글로 전환한다.

- 전체 책을 `프롤로그`부터 `에필로그`까지 30개 브런치 글로 나눈다.
- 한 글은 원칙적으로 책의 한 장을 그대로 사용한다.
- 각 글에는 2:1 가로형 삽화 3장을 넣는다.
- 주인공, 창업자, 운영자처럼 중심 인물이 등장하면 여성으로 통일한다.
- 주변 인물은 성별 표현, 나이, 직무, 체형, 피부색, 접근성 측면에서 다양하게 구성한다.
- 브런치 글은 원고를 거의 유지하되, 제목/도입/캡션만 브런치용으로 가볍게 다듬는다.

## 전체 패키지

- 브런치 원고: `posts/`
- 장별 삽화: `assets/illustrations/{글 ID}/01.png`-`03.png`
- 삽화 비율: 2:1 가로형
- 산출 매니페스트: `manifest.md`
- 남은 삽화 큐: `image_queue.json`
- 패키지 생성 스크립트: `../00_management/scripts/build_brunch_package.mjs`

## 브런치 변환 규칙

- 제목의 `N장.` 표기는 브런치 제목에서는 제거한다.
- 본문 첫 문단은 그대로 두되, 필요하면 한두 문장짜리 브런치 도입부를 앞에 붙인다.
- 책의 논리 순서는 유지한다.
- 제품명, 가격, 기관명, PoC 범위처럼 현실 계약으로 오해될 수 있는 문장은 발행 직전 확인한다.
- 이미지 안에는 텍스트를 넣지 않는다. 캡션은 브런치 본문에서 별도로 붙인다.
- 한 글은 한 장을 원칙으로 한다. 길이가 긴 장만 두 편으로 나눈다.

## 삽화 메모

- 기존 MHMbook 부별 삽화와 맞춘 흰색/연한 파란색 3D 아이소메트릭 톤을 사용한다.
- 삽화는 먼저 해당 글을 읽고, 글의 논리 전환에 맞는 맞춤 프롬프트 3개를 뽑은 뒤 생성한다.
- 3번 삽화는 메시지가 강하므로, 브런치 톤이 너무 세게 느껴지면 더 순한 버전으로 재생성한다.
- 원본 생성물은 Codex 기본 생성 폴더에 두고, 프로젝트 사용본만 `MHMbook/brunch/assets/illustrations/`에 복사했다.
- 첫 생성본 중 중심 인물이 남성처럼 보이는 이미지는 사용하지 않고 여성 주인공 기준으로 재생성한다.
